Import wizard — connect → map → dry-run preview → import

The guided importer that brings an existing backlog (Jira · Linear · GitHub Issues · CSV) into Motir work items. Four gated steps; nothing is written to the PM core until the user confirms the dry-run preview. Grounded in the 7.16.2 decision (MOTIR-938): the connector-interface source set, the full field mapping, the unmatched-value policy, the external-id-map idempotency (re-run = upsert), and the dry-run-shares-the-engine gate. Mirrors Plane (Summary→Confirm gate), Linear (multi-source→one model), and Atlassian's CSV external-id skip. Reached from the onboarding entrance "I have an existing project — import it" door and Settings › Project › Import.
