Every chart colour routes through one of these (or the 4.6.1 set) — never a raw --color-* in chart markup (mistake #20 growth pattern). They inherit the [data-theme="dark"] flip via their --color-* mapping (toggle dark to confirm) and a future data-palette. Colour is never the sole signal (finding #35): the donut carries a text legend with count + percentage per segment, the difference chart names both series in the legend + ships a data-table fallback, and the deficit/surplus fills are reinforced by which line sits on top.

Per-widget isolation: each renderer wraps its body in these states so one failure never takes the grid down. Loading = skeleton (the shape of the body to come). Empty = zero matching issues. Error = ErrorState idiom + Retry. No-access = the 6.4 per-VIEWER gate — a locked card that leaks no counts, rows, or chart shape (a workspace-shared dashboard renders for everyone, but each widget read is gated per viewer). Stale = the “Filter missing” degraded body 6.2.2 already designed (warning glyph + serif headline + one-line cause, state also carried by text — finding #35) — we INHERIT it and add only the in-grid Choose a filter reconfigure action.
