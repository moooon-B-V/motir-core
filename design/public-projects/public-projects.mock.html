You're looking at Motir, inside Motir

This is Motir's own project — the live board, roadmap, and backlog we use to build Motir. We dogfood completely: every feature here started as an issue on this board, moved through these columns, and was shipped by the same agents that turn issues into code.

A self-improving loop — and you're in it

This is also our public feedback portal, and it's not a suggestion box that goes nowhere. The bugs you report and the ideas you upvote land in Motir's triage, get planned as work items right here, and are picked up by Motir to build the next version of Motir. You're not just watching the roadmap — you're shaping it.

  1. You submit a bug or an idea — it enters our triage.
  2. We plan it as a work item on this board.
  3. Motir's coding agents turn it into a pull request.
  4. It ships — and lands as Done on this very roadmap.

Contribute

Found a bug or have an idea? Submit a request — a minute of your time feeds straight into the loop. Want to go deeper? The PM core is open source (GPL-3.0) on GitHub.

(6) Project settings — the four-level Access control (public > open > limited > private) + the shareable public link (copy / disable / rotate) + the account-required note

Project access

Who can see and act on this project. Only project admins can change this.

Public
Any signed-in Motir account — across orgs and workspaces — can view this project and submit, upvote, and comment on requests. They can't edit anything else.
Open
Any member of this workspace can view and edit.
Limited
Any member of this workspace can view and comment, but not edit.
Private
Only people added to this project can see it.
Public link
https://motir.co/p/moooon-motir
The link is a pointer to the public project — visitors still sign in to Motir to view it. Anonymous, logged-out access isn't supported. Rotating issues a new link and retires the old one; the project key (PROD) doesn't change.
