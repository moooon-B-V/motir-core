Organization token-cost dashboard

Story 7.2 · Subtask 7.2.10 design asset — built from app/globals.css tokens. The org admin’s home for token cost (all cost views live at the ORG level — Yue, locked). It SHOWS usage; it composes into the 6.10 org-admin area. Mirror = the lovart-style transparent per-model usage view (rung 1). Credits are an INTERNAL unit, never currency. Checkout / pricing / upgrade is Epic 8 and ABSENT here — only a passive “out of credits” placeholder appears. Source of truth = this HTML; see design-notes.md. Gates the 7.2.11 org cost dashboard (MOTIR-824).
