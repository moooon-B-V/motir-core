Notes. The builder is a Popover (role="dialog") anchored under the [Advanced] ToolbarButton; the combinator is the Segmented control read as a sentence; each condition row is a labelled role="group" of two Combobox triggers (field · operator — the row RENDERS the shipped 6.1.1 registry) + the operator's value editor — the MultiSelectPicker for every enum field, Input for text/number/days, the DatePicker trigger for dates — plus the remove ×. Colour flows only through --el-* (count badge / selected rows / Check = --el-accent; stale = --el-warning glyph + peach tint + strong text, never colour alone); shape via the element-semantic tokens. Filters apply live to ?filter=v1:… — no Apply button; pending rows are excluded until complete.
