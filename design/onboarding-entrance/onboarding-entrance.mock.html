Empty idea + keyboard
Start planning works with an empty idea too — the discovery chat opens and asks the first question. The idea box is a head-start, not a gate. The textarea holds focus on load; ⌘/Ctrl + Enter submits.
The Import row is a secondary but first-class control — a focusable button with its own hover / focus ring; not hidden behind a menu.
Reachability — this screen only ROUTES
Start planning → the 7.3 discovery chat, seeded with the idea (MOTIR-1462 hands off). Import → the 7.15 migrate wizard (or 7.17 for Jira / Linear / Plane), which owns connect → source-selection → index → generate.
