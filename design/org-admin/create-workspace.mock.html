Create-workspace flow

Story 6.10 · Subtask 6.10.10 design asset — built from app/globals.css tokens. Draws the surface the org menu's New workspace row opens (6.10.1 drew the row, not the dialog). A name-it Modal whose new workspace is seeded by copying a source workspace's config (6.10.2 §6e / 6.10.9) — the "looks-inherited" behaviour — plus the tier-2 first-reveal it triggers. The example is the real moooon B.V. portfolio: the organization is the company, and each workspace is a product — here the existing Motir workspace and a new Taq workspace being set up (a second moooon product), NOT departments of one product. Mirror = Jira create-project "share settings" / Linear create-team (rung 1). Billing is 7.12.5 / Epic 8 and ABSENT here. Source of truth = this HTML; see design-notes.md. Consumed by 6.10.5, visualises 6.10.9.
